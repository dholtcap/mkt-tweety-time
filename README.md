This extension adds a button on x.com which when clicked will display the delta between the time a tweet was posted and the most recent open or close of the New York Stock Exchange, as well as the time delta between posting time and the next NYSE open or close.

To install, simply download the file folder, open your chromium-based browser and open the extensions page. From there, toggle developer mode to "On", then click "Load Unpacked" and select the folder from wherever you saved it to on your machine.
